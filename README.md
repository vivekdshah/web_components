# web_components

Web Inbuilt Components  for your website.


## Features

 - Custom Header

## Getting started

To use this package, add web_components as a dependency in your pubspec.yaml file.

## Usage

Minimal example:

```dart
     HeaderWidget(
            headerTitle: 'Home',
            backgroundColor: Colors.black,
            centerHeader: true,
            headerStyle: TextStyle(color: Colors.white, fontSize: 33),
          )
```
